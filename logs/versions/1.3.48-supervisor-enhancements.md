# Versão 1.3.48 - Supervisor Enhancements & Transcription Fixes

**Data:** 15 de Abril de 2026

## O que mudou
- **Gestão de Usuários (Supervisores):** As colunas `sector_id` e `escala` foram adicionadas de forma permanente ao banco de dados, compondo agora o módulo de usuários. O formulário do sistema para o cadastro de Administradores/Supervisores (em "Configurações > Usuários") permite gerenciar essas métricas em tempo real para controle correto nas avaliações.
- **Normalização de Nomes na Transcrição:** Aprimorada a lógica de restrição para nomes chave (ex: Opentech, Mondelez, Unilever). Devido ao comportamento da transcrição com provedores modernos (Whisper ASR), casos em que os agentes mencionam "Opentech" estavam sendo confundidos (ex: "Open track", "Open desk", "Open tel"). Agora os termos possuem verificação extremamente rígida em `config/text_corrections.json` para compulsoriamente formatar corretamente esses termos.
- **Player de Áudio no Portal de Supervisor:** Incluída a mesma validação implementada previamente no painel de contestação, explicando amigavelmente quando o link do áudio é quebrado em razão da infraestrutura ser efémera no Cloud Run (e de não possuirmos integração Cloud Storage atualmente).

## Deploy
- Alterações "buildadas" e enviadas com `gcloud run deploy`.
- URL de Produção atualizada em `%100` de alocação de tráfego.
