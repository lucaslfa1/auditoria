## Paridade com original
A funcao deletada em `a6a72aa` era:
```python
def get_operators_summary_for_prompt() -> str:
    names = get_colaboradores_para_prompt()
    if not names:
        return ""
    return "\n".join(f"- {name}" for name in names)
```

A versao restaurada acrescenta o header `OPERADORES CADASTRADOS (...)` para dar contexto ao LLM (antes a lista nua era despejada apos `{known_operators}` no template) — o header e curto e nao altera o filtro de operadores.
