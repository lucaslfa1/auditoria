# Versao 1.3.38

Data: 2026-03-06

## Paleta do modo claro
- O modo claro foi reequilibrado com uma paleta unica, mais nitida e menos azulada, baseada em superficies quentes, bordas suaves e contraste de texto mais consistente.
- As sombras, gradientes e fundos do `light` foram ajustados para reduzir o aspecto lavado e deixar a hierarquia visual mais clara.

## Padrao visual unico
- O frontend passou a fixar o preset `corporativo` no runtime, evitando que presets antigos deixem o modo claro incoerente.
- O carregamento de presets visuais antigos deixou de influenciar a interface no uso diario.

## Base tecnica
- `index.css` recebeu revisao estrutural dos tokens e overrides do `theme-light`.
- Os testes frontend foram atualizados para refletir o padrao visual unico adotado.
