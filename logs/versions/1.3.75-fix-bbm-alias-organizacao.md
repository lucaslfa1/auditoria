# Versao 1.3.75 - Correcao alias BBM organizacao Huawei

**Data:** 18 de Maio de 2026
**Autores:** Lucas Afonso

## Contexto
A migracao 1.3.74 absorveu BBM por Distribuicao e removeu o setor operacional
BBM. Ficou uma regra residual em `sector_aliases`:
`organizacao_contains='bbm' -> transferencia`.

## O que mudou
- `backend/database.py`: bootstrap de `sector_aliases` passa a mapear
  `organizacao_contains='bbm'` para `distribuicao`.
- `backend/db/migration_steps/m20260518_005_fix_bbm_organization_alias.py`:
  atualiza bancos existentes, com audit_log, para alinhar organizacao Huawei
  BBM com Distribuicao.
- `backend/tests/test_bbm_distribuicao_migration.py`: adiciona guarda para a
  migration residual e para o alias de bootstrap.

## Validacao
- Migration aplicada no banco: `m20260518_005_fix_bbm_organization_alias`.
- `resolve_canonical_sector(organizacao='N2 BBM Huawei')` retorna
  `distribuicao`.
- `audit_alerts`: 67 alertas ativos; zero alertas ou setor `bbm`.
