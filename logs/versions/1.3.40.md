## Objetivo
- Consolidar a refatoracao frontend/backend, registrar a validacao executada e publicar a microversao com log formal.

## Bugs Corrigidos
- **BUG-040: Gerador de log nao preenchia `base_app_version` (baixa)** - O script `scripts/new-log-version.mjs` mantinha o placeholder `x.y.z` no YAML porque so inseria o campo quando ele nao existia no template. A substituicao explicita do placeholder foi adicionada.

## Mudancas
- Consolidada a migracao do frontend para estrutura feature-based com `src/features/` e `src/shared/`, removendo componentes legados de `src/components/`, hooks antigos de `src/hooks/` e barrels obsoletos.
- Consolidada a reorganizacao do backend com `routers/`, `repositories/`, `db/` e `persistence.py`, reduzindo o acoplamento direto da aplicacao a `database.py`.
- Endurecida a superficie HTTP do backend com CORS explicito, headers de seguranca e rate limit configuravel no login.
- Scripts manuais que pareciam testes automatizados foram movidos para `backend/scripts/manual_*`, reduzindo confusao na descoberta da suite.
- Registrada a revisao tecnica em `docs/reviews/refactoring-review-2026-03-07.md` e alinhada a versao da aplicacao para `1.3.40`.

## Arquivos Alterados
- `src/features/` - nova organizacao por dominio no frontend.
- `src/shared/` - fonte canonica para componentes, hooks, tipos e cliente HTTP compartilhados.
- `backend/persistence.py` - facade runtime para compatibilidade com a camada legada.
- `backend/main.py` - roteamento principal, CORS explicito e headers de seguranca.
- `backend/routers/auth.py` - login/logout/sessao com rate limit configuravel.
- `backend/repositories/` - acesso a dados modularizado por agregado.
- `backend/db/` - conexao, schema runtime e migracoes versionadas.
- `backend/tests/test_http_security.py` - cobertura para CORS, headers e rate limiting.
- `backend/tests/test_saved_files_repository.py` - cobertura dedicada de repository.
- `docs/reviews/refactoring-review-2026-03-07.md` - avaliacao tecnica consolidada da refatoracao.
- `scripts/new-log-version.mjs` - correcao do preenchimento de `base_app_version`.
- `package.json` - versao atualizada para `1.3.40`.
