## BUG-019: fallback silencioso para `localhost` em `get_database_url()`

**Arquivo:** `backend/db/connection.py:79-89`
```python
def get_database_url() -> str:
    url = os.getenv("DATABASE_URL", "").strip()
    if not url:
        url = "postgresql://postgres:postgres@localhost:5432/auditoria"  # <-- fallback silencioso
    ...
```

**Risco:** se `DATABASE_URL` nao for setada (typo no `.env`, env var nao propagada), a aplicacao conecta num PostgreSQL local sem nenhum aviso. Em ambientes onde o local existe (dev workstation com PG instalado), os dados sao gravados/lidos do banco errado.

**Fix recomendado:** levantar `RuntimeError` em producao quando `DATABASE_URL` for vazia, ou ao menos logar com `logger.error`.
