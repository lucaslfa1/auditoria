## 1. Resumo Executivo

Este documento integra os **198 funcionários** consolidados com os **critérios de auditoria** definidos pela OpenTech, criando um mapa de referência rápida para:

- Identificar quais critérios auditam cada setor
- Consultar quais operadores estão sob supervisão de determinada pessoa
- Entender as regras de zeragem (critérios não-negociáveis) por setor
- Planejar auditorias por escala (turno)

## 5. Cronograma de Auditoria Mensal

**Baseado em Manual Técnico Qualidade OpenTech:**

```
SEMANA 1 (Dias 1-5):
├─ Confirmação da Listagem de Operadores
│  ├─ Prazo para gestores: 2 turnos
│  ├─ Validação de: nome, matrícula, ID WEON, turno, supervisão
│  └─ Atualização de férias, afastados, funções não-auditáveis
│
├─ Requisitos para Auditoria (todos os operadores)
│  ├─ Trabalhado mínimo 7 dias no mês
│  └─ Mínimo 7 ligações auditáveis no período

SEMANA 2-3 (Dias 8-21):
├─ Período Efetivo de Auditoria
│  └─ Ferramenta: Arquivo Excel (matrículas, alertas, ligações)

SEMANA 4 (Dias 22-28):
├─ Finalização da Auditoria
├─ Prazo de Entrega e Retificação
│  ├─ Última semana do mês: auditoria finalizada
│  └─ Relatos devem ser: detalhados, completos, com confirmação

DIA 1-2 DO MÊS SEGUINTE:
├─ Checagem Final
└─ Fechamento Final
```

## 6. Checklist para Implementação da Auditoria Integrada

- [ ] **Confirmação de Operadores (Semana 1)**
  - [ ] Validar dados de cada operador na planilha
  - [ ] Sinalar férias, afastados, funções não-auditáveis
  - [ ] Deadline: 2 turnos

- [ ] **Seleção de Chamadas (Semana 2-3)**
  - [ ] Buscar por filtros no sistema de telefonia (Huawei/Weon)
  - [ ] 2 ligações por operador por setor
  - [ ] Alertas específicos por setor (ver matriz acima)
  - [ ] Incluir relatos detalhados com: nome, telefone, confirmação de senha

- [ ] **Auditoria via Sistema (Inteligência Artificial)**
  - [ ] Importar áudios para pasta `Ligações/{SETOR}/`
  - [ ] Nomenclatura: `[ALERTA]_[OPERADOR].[WAV/MP3]`
  - [ ] Sistema infere automaticamente: setor, alerta, operador
  - [ ] IA avalia baseado em critérios com pesos (docs/references/auditoria/criterios-auditoria-opentech.md)
  - [ ] Aplica regras de zeragem conforme setor

- [ ] **Retificação e Fechamento**
  - [ ] Gestores processam feedback conforme resultado
  - [ ] Operadores reauditados se necessário
  - [ ] Fechamento nos 2 primeiros dias do mês seguinte
