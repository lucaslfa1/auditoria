# Manual para Gestores

Data de referencia: 2026-04-08

## Objetivo

Este material consolida o funcionamento do sistema em linguagem operacional para gestao.

Ele foi criado para:
- explicar o fluxo principal do sistema;
- deixar claras as regras de negocio ja formalizadas;
- registrar o que ja foi revisado com seguranca;
- servir como base para decisao, acompanhamento e alinhamento com a operacao.

## Escopo atual

Este manual reflete:
- visao geral do sistema;
- fluxo operacional principal;
- comportamento atual do modulo de triagem;
- regras de negocio ja confirmadas;
- status operacionais que a gestao precisa acompanhar.

## Modulos ja revisados formalmente

- triagem

Os demais modulos podem ser adicionados nesta pasta conforme avancarem as revisoes.
