# Referencias Canonicas

Esta pasta guarda fontes externas e operacionais que embasam regras do sistema.
Use esta pasta para documentos fonte; relatorios datados continuam em
`docs/reports/` e conhecimento RAG curado continua em `rag/`.

## Subpastas

- `auditoria/`: criterios, qualidade, calibracao e regras nao negociaveis.
- `automacao/`: manuais e referencias de RPA/automacao.
- `operacional/`: glossarios e referencias de operacao.

## Regra

Nao copie um documento desta pasta para outro local do projeto. Referencie o
caminho canonico.
