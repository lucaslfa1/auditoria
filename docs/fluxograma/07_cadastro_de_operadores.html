Processo 7: Cadastro e Gestão de Operadores

Para que o robô do sistema consiga baixar ligações automaticamente, o operador precisa estar devidamente cadastrado no banco de dados e marcado como ativo. Este fluxograma descreve os passos para cadastrar ou importar novos colaboradores.

graph TD Start([Painel de Operadores]) --> Choice{Como cadastrar?} Choice -- 1. Novo Colaborador (Formulário) --> Form[Preencher Nome, Supervisor, Setor e Escala] Choice -- 2. Importar Planilha (Em lote) --> ExcelImport[Importar arquivo Excel de operadores] Form --> FieldHuawei[Cadastrar Matrícula e ID Huawei da telefonia] ExcelImport --> FieldHuawei FieldHuawei --> CheckActive{Marcar como Ativo e Auditável?} CheckActive -- Não --> SkipSync[Operador é cadastrado, mas ignorado pelo robô da telefonia] CheckActive -- Sim --> EnableSync[Robô inclui o operador na busca de ligações diária] EnableSync --> End([Operador Habilitado para Coleta]) SkipSync --> End style Start fill:#0284c7,stroke:#0369a1,stroke-width:2px,color:#fff style End fill:#0284c7,stroke:#0369a1,stroke-width:2px,color:#fff style Choice fill:#f59e0b,stroke:#d97706,stroke-width:2px,color:#fff style CheckActive fill:#f59e0b,stroke:#d97706,stroke-width:2px,color:#fff style EnableSync fill:#10b981,stroke:#059669,stroke-width:2px,color:#fff style SkipSync fill:#ef4444,stroke:#dc2626,stroke-width:2px,color:#fff
O que acontece em termos simples:
  • ID Huawei: Este é o código do operador dentro do sistema de telefonia. Sem ele preenchido, o robô não tem como cruzar os dados da ligação, e o operador "some" da coleta diária.
  • Marcação Auditável: Permite desativar temporariamente a auditoria de um operador (como em férias ou afastamento) sem precisar apagá-lo do histórico.
  • Importação em Lote: Ideal para cadastros massivos ou atualizações periódicas de equipe usando planilhas.
